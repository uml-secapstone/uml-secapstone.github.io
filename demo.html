Chat with a Language Model

A large language model (LLM) is a type of AI model that is trained on massive datasets of text to then generate vocabulary, grammar, and context. A well-known example is ChatGPT, but we have tailored our LLM from Facebook's OLlama. We currently host our LLM on a Raspberry Pi device as the API endpoint.

Neural Network Simulation

A neural network that consists of layers of "neurons" that process information. In classification, a neural network takes input data, processes it through layers, then decides which category it belongs to. For instance, if you give it a picture of a keyboard or a mouse, it learns patterns from past images and predicts whether the new image is a keyboard or mouse.

In machine learning, an epoch means going through the entire dataset once to recognize patterns. Batch sizes break up the number of samples processed at once. The learning rate of a model influences how quickly and effectively the model learns. A high learning rate can lead to faster learning, but may become unstable while a low learning rate can lead to slow training. Lastly, the loss result describes a measure of how inaccurate the model is - the goal is to reduce the loss so the model makes fewer mistakes.
