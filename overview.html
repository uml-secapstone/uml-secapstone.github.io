Project Overview

Adversarial Attacks

One adversarial attack we have implemented is the C&W attack, which imperceptibly modifies images. Models with publicly available parameters are vulnerable.

Language Models

We have implemented this adversarial attack towards language models, especially with a chatbox platform. Our goal is to trick the language models in returning responses for "unethical questions".

Anomaly Detection Model

Our implementation of an anomaly detection model aims to recognize adversarial attacks in ML, improving the security and reliability of services that use AI.
